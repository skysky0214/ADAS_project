# 5.11 커밋 메모

## 현재 진행상황

- workspace 구조를 `app/core`, `app/pipeline`, `app/perception`, `app/tracking`, `app/prediction`, `app/bridge`, `app/control`, `tools` 중심으로 정리했다.
- OpenPCDet DSVT 인지 모델 checkpoint를 perception adapter로 연결했다.
- `app/main.py`에서 `/lidar_points`를 받아 DSVT 인지, 보행자 필터링, tracking, SR-LSTM 예측까지 live pipeline으로 실행한다.
- planner-facing snapshot 생성을 `TrackedPedestrian + PredictedTrajectory` 기준으로 정리했다.
- RViz2용 `/adas/tracking_markers`를 publish한다.
  - 3D box, track ID, 지나온 경로, 예측 경로, TTC warning text 표시
- TTC warning adapter를 추가했다.
  - 실제 제어 command는 만들지 않고, 경고 level과 감속도 후보만 계산한다.

## 남은 태스크

- tracking ID 안정성 개선
- 자차 이동 경로/ego motion 가정 확정
- 하드웨어 팀과 실제 제어 command interface 확정
- TTC warning을 실제 vehicle control 단계와 연결
- rosbag 장시간 실행으로 latency와 frame drop 확인
